<h1 align="center">📊 Excel Sales Analysis Dashboard</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents an interactive <b>Sales Analysis Dashboard</b> developed using Microsoft Excel.
The dashboard provides meaningful insights into sales performance, profit trends, customer behavior,
and product performance. It helps businesses analyze data efficiently and make informed decisions
through visual reports and interactive charts.
</p>

<h2>🎯 Problem Statement</h2>
<p>
Businesses generate large amounts of sales data daily, making it difficult to manually analyze trends
and identify important insights. This project provides a user-friendly dashboard that simplifies sales
analysis by visualizing KPIs, revenue trends, profit performance, and category-wise sales data.
The dashboard helps in monitoring business growth and improving decision-making.
</p>
